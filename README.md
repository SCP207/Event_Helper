# Event_Helper_SL_EXILED
This plugin provides 5 useful commands for events:

- infinite ammo

- infinite ammo in guns

- stop waves

- spawn waves giving items

- spawn waves giving effects

- stop teslas

- Prevent people from spawning with items

## Permissions:

- eh.infammo

- eh.wavesenabled

- eh.wavegiveitems

- eh.wavegiveeffects

- eh.stopteslas

- eh.noitemspawn
